# Global Docs (PDMS)

Documentacao global partilhada por todo o repositorio.

## Conteudo

- `GATEWAY_ACCESS_MODEL.md`: detalhes do gateway (auth, sessao, launcher)
- `PDMS_APP_INTEGRATION.md`: contrato de integracao entre gateway e sub-apps

## Objetivo

Centralizar decisoes de arquitetura e contratos de comunicacao para evitar divergencia entre apps.
